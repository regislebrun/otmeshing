# otmeshing module

Installation
  Silent install can be done with the switch /S (e.g. openturns-otmeshing-0.0-py27.exe /S).
  To test otmeshing module installation, in an OpenTURNS python console type:
    import otmeshing
    print otmeshing.__version__

Uninstallation
  otmeshing module will be automatically uninstalled at OpenTURNS removal.
  otmeshing module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
